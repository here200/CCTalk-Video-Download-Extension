# CCTallk-Video-Download-Extension

原项目： [here200/CCTallk-Video-Download: Cctalk视频下载工具 (github.com)](https://github.com/here200/CCTallk-Video-Download)

感谢大家对 原项目的支持，那是我半年多前写下的代码。那时写下来后，直接上传到了**github**。

原项目存在一些问题。

1. 采用类的方式，只是为了数据共享。
2. 各个模块全部耦合在一个类中，不利于扩展。
3. 用户在使用该代码时，可操作空间不高。

为此，我决定了重构原项目。在此之前，我还是要感谢大家，对原项目的支持。因为这是一个比较简单的爬虫项目。爬虫新手预计也能写出来。同时，这是截至目前(2022-10-20)，我收获 **Star数** 和 **Fork 数** 最高的项目。

我并不想将重构后的代码直接覆盖到 原来的项目中。因为这可能造成别人理解上的困难。与其这样，还不如将它们分开。【原项目能正常使用，不想改动它】。

### 补充

- 在重构前，我也犹豫过，因为原项目也就90行(准确来说，也没有90行)，没有太大的必要进行重构。为了更好的扩展性，所以选择了重构。
- 在这个项目中，我不想介绍有关它是如何使用的。因为它的使用方式，只要你了解 原项目怎么使用的，这个项目，我相信你能知道如何使用。
  - 还是一样，在进行下载视频前，请确保在 main.py 同一路径下，是否有名为 data 的文件夹。如果没有，请手动添加一个。
- 新特性
  1. 默认会给每个下载的视频添加序号【可选，也可以不添加】
  2. 已经下载过的视频，不会重新下载【会直接跳过，避免重复下载视频】
